07 · set

Sets y sus métodos

Un set es una colección sin duplicados y sin orden. Es muy útil para eliminar repetidos de una lista o verificar pertenencia rápidamente.

sets_intro.py
# Eliminar duplicados de una lista
lista = [1, 2, 2, 3, 3, 4]
sin_duplicados = list(set(lista))
print(sin_duplicados)   # [1, 2, 3, 4]

# Crear un set
frutas = {"manzana", "pera", "uva"}
print("manzana" in frutas)   # True (muy rápido)

.remove(x)Elimina x. KeyError si no existes.remove(2){1,3}
.discard(x)Elimina x sin error si no existes.discard(99)sin error
.union(otro)Elementos de ambos sets (sin duplicados){1,2}.union({2,3}){1,2,3}
.intersection(otro)Solo elementos en AMBOS sets{1,2,3}.intersection({2,3,4}){2,3}
.difference(otro)Elementos en el primero pero NO en el segundo{1,2,3}.difference({2,3}){1}

08 · iteradores

Iteradores especiales

Funciones built-in que hacen los bucles más poderosos y expresivos. Estos son los que aparecen frecuentemente en ejercicios como el que mencionaste.

FunciónDescripciónEjemploResultado
LOS MÁS IMPORTANTES
enumerate(iter, start) Recorre un iterable dando (índice, valor). start define desde qué número empieza el índice (por defecto 0) for i, v in enumerate(["a","b"], 1) i=1,v="a" / i=2,v="b"
zip(a, b) Empareja elementos de dos iterables en tuplas. Se detiene en el más corto list(zip([1,2],["a","b"])) [(1,'a'),(2,'b')]
map(fn, iter) Aplica una función a cada elemento. Retorna iterador list(map(int, ["1","2","3"])) [1, 2, 3]
filter(fn, iter) Filtra elementos donde fn retorna True list(filter(lambda x: x>2, [1,2,3,4])) [3, 4]

enumerate() — el más común en ejercicios

enumerate_ejemplo.py
frutas = ["manzana", "pera", "uva"]

# Sin enumerate — necesitas manejar el índice manualmente
i = 0
for fruta in frutas:
    print(i, fruta)
    i += 1

# Con enumerate — más limpio
for i, fruta in enumerate(frutas):
    print(i, fruta)

# Con start=1 para empezar desde 1 en lugar de 0
for i, fruta in enumerate(frutas, 1):
    print(f"{i}. {fruta}")
Output (último bloque)
1. manzana
2. pera
3. uva

zip() — combinar dos listas

zip_ejemplo.py
nombres = ["Ana", "Luis", "Carlos"]
notas   = [85, 92, 78]

for nombre, nota in zip(nombres, notas):
    print(f"{nombre}: {nota}")
Output
Ana: 85
Luis: 92
Carlos: 78
💡

enumerate vs zip: usa enumerate cuando necesitas el índice de un solo iterable. Usa zip cuando quieres recorrer dos listas en paralelo al mismo tiempo.
