01

¿Qué es una condición?

Una condición es una expresión que Python evalúa y que solo puede tener dos resultados: True (verdadero) o False (falso). En base a ese resultado, el programa decide qué código ejecutar.

Es como un semáforo: si la luz es verde → avanza. Si no → espera. El programa hace lo mismo con datos.

condicion.py
# Estas expresiones devuelven True o False
print(5 > 3)      # True
print(5 == 3)     # False
print("a" in "hola") # True
print(10 != 10)    # False
Output
True
False
True
False
02

Anatomía completa

Un bloque condicional puede tener tres partes. Solo if es obligatorio — elif y else son opcionales:

if condición :
instrucciones si es True
elif otra condición :
instrucciones si esta es True
elif otra condición más :
puedes tener tantos elif como necesites
else :
instrucciones si ninguna condición fue True
if — primera condición
elif — condición alternativa (0 o más)
else — caso por defecto
condición — expresión True/False
body — debe estar indentado (4 espacios)
💡

Orden importa: Python evalúa de arriba hacia abajo. En cuanto encuentra una condición True, ejecuta ese bloque y se salta todos los demás. El else solo corre si ninguna condición anterior fue True.

logicos.py
edad = 20
tiene_dni = True

# and: ambas deben ser True
if edad >= 18 and tiene_dni:
    print("Puede votar")

# or: al menos una True
if edad < 12 or edad >= 65:
    print("Descuento especial")

# not: invierte
if not tiene_dni:
    print("Necesita DNI")
Output
Puede votar
06

Ejemplos básicos

Solo if

solo_if.py
temperatura = int(input("Temperatura: "))

if temperatura > 35:
    print("Hace mucho calor")

print("Programa terminado")  # esto siempre se ejecuta
Output (si ingresa 40)
Hace mucho calor
Programa terminado

if / else

if_else.py
nota = int(input("Nota (0-100): "))

if nota >= 60:
    print("Aprobado ✓")
else:
    print("Reprobado ✗")
Output (si ingresa 45)
Reprobado ✗

if / elif / else

if_elif_else.py
nota = int(input("Nota (0-100): "))

if nota >= 90:
    print("A — Excelente")
elif nota >= 80:
    print("B — Muy bien")
elif nota >= 70:
    print("C — Bien")
elif nota >= 60:
    print("D — Suficiente")
else:
    print("F — Reprobado")
Output (si ingresa 75)
C — Bien
✅

Tip de eficiencia: en una cadena elif, no necesitas repetir la parte izquierda del rango. Si llegaste al elif nota >= 80, Python ya sabe que nota < 90 porque el if anterior falló.

07

Condicionales anidados

Un if dentro de otro if. Úsalos cuando una condición depende de que otra ya se haya cumplido:

anidados.py
edad = int(input("Edad: "))
pais = input("País: ").lower()

if edad >= 18:
    print("Es mayor de edad")
    if pais == "peru":
        print("Puede votar en Perú")
    else:
        print("Verificar reglas de su país")
else:
    print("Es menor de edad, no puede votar")
Output (edad=20, país=peru)
Es mayor de edad
Puede votar en Perú
⚠️

Cuidado con el exceso: más de 2-3 niveles de anidación dificultan leer el código. A veces es mejor usar and para combinar condiciones en una sola línea.

08

Casos especiales

Operador ternario
Un if/else en una sola línea. Útil para asignaciones simples.
resultado = "par" if n % 2 == 0 else "impar"

# equivale a:
if n % 2 == 0:
  resultado = "par"
else:
  resultado = "impar"
match / case (Python 3.10+)
Alternativa moderna a muchos elif. Como el switch de otros lenguajes.
match opcion:
  case "1": print("uno")
  case "2": print("dos")
  case _: print("otro")
Condición con rango
Python permite encadenar comparaciones directamente, sin and.
# Forma larga
if n >= 18 and n <= 65:

# Forma corta (solo Python)
if 18 <= n <= 65:
Truthy / Falsy
Python evalúa algunos valores directamente como True o False sin necesidad de comparar.
# Falsy: 0, "", [], None, False
# Truthy: cualquier otro valor

nombre = ""
if nombre: # False porque es vacío
  print(nombre)

Operador ternario — ejemplo completo

ternario.py
edad = int(input("Edad: "))

# En una línea:
estado = "mayor" if edad >= 18 else "menor"
print(f"Es {estado} de edad")
Output (si ingresa 15)
Es menor de edad

Truthy / Falsy — ejemplo completo

truthy.py
nombre = input("Tu nombre: ").strip()

if nombre:           # True si nombre no está vacío
    print(f"Hola, {nombre}")
else:                 # False si nombre == ""
    print("No ingresaste un nombre")

Errores comunes

errores.py
# ❌ ERROR 1: = en vez de ==
if x = 5:   # SyntaxError
if x == 5:  # ✅ correcto

# ❌ ERROR 2: olvidar los dos puntos
if x > 0    # SyntaxError
if x > 0:   # ✅ correcto

# ❌ ERROR 3: indentación incorrecta
if x > 0:
print("positivo")  # IndentationError
if x > 0:
    print("positivo")  # ✅ 4 espacios

# ❌ ERROR 4: elif después de else
else:
    print("...")
elif x == 0:  # SyntaxError — else va siempre al final
    print("...")

# ❌ ERROR 5: comparar tipos distintos
edad = input("Edad: ")    # edad es str
if edad >= 18:            # TypeError: str vs int
    print("mayor")
edad = int(input("Edad: "))  # ✅ convertir primero
