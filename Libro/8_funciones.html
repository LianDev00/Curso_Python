01

¿Qué son las funciones?

Una función es un bloque de código con nombre que puedes escribir una vez y usar cuantas veces necesites. En lugar de repetir el mismo código en varios lugares, lo metes en una función y simplemente la llamas.

Piénsalo como una máquina: le metes datos (parámetros), la máquina los procesa, y te devuelve un resultado (return). No necesitas saber cómo funciona por dentro cada vez que la usas.

💡

Sin funciones: copias y pegas código. Con funciones: escribes una vez, usas siempre. Si algo cambia, solo modificas un lugar.

sin_funciones.py vs con_funciones.py
# ❌ Sin funciones — código repetido
nombre1 = "Ana"
print(f"Hola, {nombre1}! Bienvenida.")

nombre2 = "Luis"
print(f"Hola, {nombre2}! Bienvenida.")

# ✅ Con funciones — una vez, muchos usos
def saludar(nombre):
    print(f"Hola, {nombre}! Bienvenido.")

saludar("Ana")
saludar("Luis")
saludar("Carlos")
Output
Hola, Ana! Bienvenido.
Hola, Luis! Bienvenido.
Hola, Carlos! Bienvenido.
02

Anatomía de una función

def nombre_funcion ( param1 , param2 ):
instrucciones del cuerpo
return valor_a_devolver
def — palabra reservada
nombre — identificador de la función
parámetros — datos de entrada (opcionales)
cuerpo — código indentado 4 espacios
return — valor que devuelve (opcional)
⚠️

Definir ≠ Ejecutar. Escribir def saludar(): solo crea la función. Para que corra debes llamarla: saludar().

03

Tipos de funciones

Sin parámetros
No recibe datos externos. Siempre hace lo mismo.
def saludar():
  print("¡Hola!")

saludar()
Con parámetros
Recibe datos para personalizar su comportamiento.
def saludar(nombre):
  print(f"Hola {nombre}")

saludar("Ana")
Con return
Devuelve un valor que puedes guardar o usar.
def sumar(a, b):
  return a + b

resultado = sumar(3, 4)
Lambda
Función anónima de una sola línea para casos simples.
doble = lambda x: x * 2

print(doble(5)) # 10
04

Parámetros y argumentos

Parámetro es la variable que defines en el def. Argumento es el valor que pasas al llamar la función. Son la misma cosa vista desde dos ángulos distintos.

Parámetros con valor por defecto

default_params.py
# Si no pasas el argumento, usa el valor por defecto
def saludar(nombre, saludo="Hola"):
    print(f"{saludo}, {nombre}!")

saludar("Ana")              # usa "Hola" por defecto
saludar("Luis", "Buenas")   # sobreescribe el defecto
Output
Hola, Ana!
Buenas, Luis!

Argumentos con nombre (keyword arguments)

keyword_args.py
def describir(nombre, edad, ciudad):
    print(f"{nombre}, {edad} años, de {ciudad}")

# Puedes pasarlos en cualquier orden si usas el nombre
describir(ciudad="Lima", nombre="Lian", edad=26)
Output
Lian, 26 años, de Lima
05

Return — devolver valores

return termina la función y devuelve un valor al lugar donde fue llamada. Sin return, la función devuelve None automáticamente.

return_ejemplos.py
# Return simple
def cuadrado(n):
    return n ** 2

resultado = cuadrado(5)
print(resultado)   # 25

# Return múltiple
def min_max(lista):
    return min(lista), max(lista)

minimo, maximo = min_max([3, 1, 7, 2])
print(minimo, maximo)   # 1 7

# Return como salida anticipada
def dividir(a, b):
    if b == 0:
        return "Error: no se puede dividir entre 0"
    return a / b

print(dividir(10, 2))   # 5.0
print(dividir(10, 0))   # Error: no se puede dividir entre 0
Output
25
1 7
5.0
Error: no se puede dividir entre 0
💡

print vs return: print() muestra algo en pantalla pero no devuelve nada útil. return devuelve el valor para que puedas guardarlo y usarlo después. Usa return cuando necesites el resultado para algo más.

06

Funciones Lambda

Una función lambda es una función anónima de una sola expresión. Es útil para funciones simples que no necesitan nombre propio.

lambda.py
# Sintaxis: lambda parámetros : expresión

# Equivalencia
def doble(x): return x * 2
doble_lambda = lambda x: x * 2

print(doble(5))         # 10
print(doble_lambda(5))  # 10

# Lambda con múltiples parámetros
sumar = lambda a, b: a + b
print(sumar(3, 4))   # 7

# Lambda con condicional
par_o_impar = lambda n: "par" if n % 2 == 0 else "impar"
print(par_o_impar(7))  # impar
Output
10
10
7
impar
⚠️

¿Cuándo usar lambda? Solo para expresiones simples de una línea. Si necesitas múltiples líneas, usa def normal. Las lambdas no mejoran la legibilidad cuando son complejas.

Función que llama a otra función

funciones_compuestas.py
def es_par(n):
    return n % 2 == 0

def filtrar_pares(lista):
    resultado = []
    for n in lista:
        if es_par(n):           # llama a es_par()
            resultado.append(n)
    return resultado

numeros = [1, 2, 3, 4, 5, 6, 7, 8]
print(filtrar_pares(numeros))
Output
[2, 4, 6, 8]

Buenas prácticas

01
Una función, una responsabilidad
Cada función debe hacer solo una cosa. Si su nombre necesita un "y", probablemente debería ser dos funciones.
02
Nombres descriptivos
Usa verbos: calcular_promedio(), validar_email(), convertir_celsius(). Evita f1() o hacer_cosa().
03
Máximo 20 líneas
Si una función es muy larga, divídela en funciones más pequeñas. Más fácil de leer y de depurar.

Siempre return explícito
Si una función debe devolver algo, usa return explícito. No dependas del None implícito.

Agrega una línea de descripción al inicio: """Calcula el promedio de una lista."""
06
Llama a main() al final
Define todas tus funciones primero, luego llama a main() al final del archivo. Es el patrón estándar de CS50P.
estructura_cs50p.py
def main():
    nombre = input("Nombre: ")
    resultado = procesar(nombre)
    print(resultado)

def procesar(texto):
    return texto.strip().title()

# Llamar main al final — patrón CS50P
if __name__ == "__main__":
    main()

Calculadora de IMC

Crea una función calcular_imc(peso, altura) que reciba el peso en kg y la altura en metros, calcule el IMC y devuelva la categoría correspondiente como string.

🎯 Reto

La función debe devolver tanto el valor del IMC como la categoría en una sola llamada. Luego crea una función main() que pida los datos al usuario y llame a calcular_imc().

💡 Pista de diseño

Usa return imc, categoria para devolver dos valores a la vez. Al llamar la función, desempácalos: imc, cat = calcular_imc(peso, altura). Recuerda: IMC = peso / altura².

📋 Tabla de análisis
IdentificadorDescripciónTipoFunciónRestricción

alturaAltura en metrosfloatParámetroMayor que 0

categoriaClasificación del IMCstrReturnBajo / Normal / Sobrepeso / Obesidad
SOLUCIÓN
imc.py
def main():
    peso = float(input("Peso (kg): "))
    altura = float(input("Altura (m): "))
    imc, categoria = calcular_imc(peso, altura)
    print(f"IMC: {imc:.2f} — {categoria}")

def calcular_imc(peso, altura):
    imc = peso / (altura ** 2)
    if imc < 18.5:
        categoria = "Bajo peso"
    elif imc < 25:
        categoria = "Normal"
    elif imc < 30:
        categoria = "Sobrepeso"
    else:
        categoria = "Obesidad"
    return imc, categoria

if __name__ == "__main__":
    main()

Conversor de Temperaturas

Crea tres funciones: celsius_a_fahrenheit(c), fahrenheit_a_celsius(f) y main(). El programa debe preguntar al usuario qué conversión quiere hacer y en qué dirección.

🎯 Reto

Cada función de conversión solo hace el cálculo y retorna el resultado — no imprime nada. Todo el input/output se maneja en main(). Usa un menú con while True para repetir hasta que el usuario elija salir.

💡 Pista de diseño

Fórmulas: F = (C × 9/5) + 32 y C = (F - 32) × 5/9. Usa input().strip().lower() para normalizar la opción del usuario.

SOLUCIÓN
temperatura.py
def main():
    while True:
        print("\ncf → Celsius a Fahrenheit")
        print("fc → Fahrenheit a Celsius")
        print("salir → Terminar")
        opcion = input("Opción: ").strip().lower()

        if opcion == "salir":
            break
        elif opcion == "cf":
            t = float(input("Celsius: "))
            print(f"= {celsius_a_fahrenheit(t):.2f}°F")
        elif opcion == "fc":
            t = float(input("Fahrenheit: "))
            print(f"= {fahrenheit_a_celsius(t):.2f}°C")
        else:
            print("Opción inválida")

def celsius_a_fahrenheit(c):
    return (c * 9 / 5) + 32

def fahrenheit_a_celsius(f):
    return (f - 32) * 5 / 9

if __name__ == "__main__":
    main()

Verificador de Palíndromo

Crea una función es_palindromo(texto) que reciba una cadena y retorne True si es palíndromo o False si no lo es. Debe ignorar mayúsculas y espacios.

🎯 Reto

La función debe limpiar el texto antes de comparar: quitar espacios y convertir a minúsculas. No uses [::-1] directamente — construye la comparación limpiando primero el texto.

💡 Pista de diseño

def main():
    texto = input("Ingresa una palabra o frase: ")
    if es_palindromo(texto):
        print(f'"{texto}" ES palíndromo ✓')
    else:
        print(f'"{texto}" NO es palíndromo ✗')

def es_palindromo(texto):
    limpio = texto.lower().replace(" ", "")
    return limpio == limpio[::-1]

Generador de Contraseña Segura

Crea una función validar_password(password) que verifique si una contraseña es segura y retorne una lista con los errores encontrados. Si la lista está vacía, la contraseña es válida.

password_validator.py
def main():
    password = input("Ingresa una contraseña: ")
    errores = validar_password(password)

    if len(errores) == 0:
        print("✓ Contraseña segura")
    else:
        print("✗ Contraseña inválida:")
        for error in errores:
            print(f"  - {error}")

def validar_password(password):
    errores = []
    if len(password) < 8:
        errores.append("Mínimo 8 caracteres")
    if " " in password:
        errores.append("No puede tener espacios")
    if not any(c.isupper() for c in password):
        errores.append("Al menos una mayúscula")
    if not any(c.isdigit() for c in password):
        errores.append("Al menos un número")
    return errores

if __name__ == "__main__":
    main()

Cifrado César con Funciones

Refactoriza el cifrado César que hiciste antes en funciones. Crea cifrar(texto, n) que cifre el texto y descifrar(texto, n) que lo descifre. Ambas deben usar una función auxiliar interna.

🎯 Reto

Crea una tercera función desplazar(texto, n) que haga el trabajo real. cifrar() la llama con +n y descifrar() la llama con -n. Así evitas duplicar código.

💡 Pista de diseño

La función desplazar(texto, n) recorre cada carácter: si es letra usa chr(ord(c) + n), si no lo deja igual. cifrar y descifrar son solo envoltorios que llaman a desplazar con signo positivo o negativo.

📋 Tabla de análisis

SOLUCIÓN
cesar_funciones.py
def main():
    texto = input("Texto: ").lower()
    n = int(input("Desplazamiento: "))
    cifrado = cifrar(texto, n)
    descifrado = descifrar(cifrado, n)
    print(f"Original:   {texto}")
    print(f"Cifrado:    {cifrado}")
    print(f"Descifrado: {descifrado}")

def desplazar(texto, n):
    resultado = ""
    for c in texto:
        if c.isalpha():
            resultado += chr(ord(c) + n)
        else:
            resultado += c
    return resultado

def cifrar(texto, n):
    return desplazar(texto, n)    # desplaza hacia adelante

def descifrar(texto, n):
    return desplazar(texto, -n)   # desplaza hacia atrás

if __name__ == "__main__":
    main()
