01 · Concepto

¿Qué es el slicing?

El slicing (en español: "rebanado") te permite extraer una parte de una cadena de texto indicando desde qué posición hasta qué posición quieres cortar. Como rebanar una barra de pan: dices desde qué rebanada hasta cuál.

En Python, cada carácter de un string tiene una posición (índice) que empieza desde 0, no desde 1.

La palabra "PYTHON" en memoria:

-6
Y
1
-5

H

↑ índice positivo     ↑ índice negativo (desde el final)
02 · Sintaxis

Cómo se escribe

La forma general es: cadena[inicio : fin : paso]

El inicio es incluido, el fin NO. Si omites alguno, Python usa el valor por defecto.

Código Resultado Descripción
palabra[0:3] "PYT" Desde índice 0 hasta 2 (el 3 no se incluye)
palabra[2:] "THON" Desde índice 2 hasta el final
palabra[:3] "PYT" Desde el inicio hasta índice 2
palabra[-3:] "HON" Los últimos 3 caracteres
palabra[::2] "PTO" Cada 2 caracteres (paso = 2)
palabra[::-1] "NOHTYP" La palabra al revés (paso = -1)
palabra[1] "Y" Un solo carácter (sin dos puntos)

El índice fin nunca se incluye en el resultado. palabra[0:3] da los caracteres 0, 1 y 2 — pero no el 3.

03 · Ejemplos

Ejemplos de código

Ejemplo 1 — Extraer parte de una palabra:

ejemplo_1.py 
nombre = "Alexander"

print(nombre[0:4])   # primeras 4 letras
print(nombre[4:])    # desde la 5ta en adelante
print(nombre[-3:])   # últimas 3 letras
Output
Alex
ander
der

Ejemplo 2 — Invertir una palabra:

ejemplo_2.py 
palabra = "hola"
invertida = palabra[::-1]
print(invertida)
Output
aloh

Ejemplo 3 — Usar slicing con input del usuario:

ejemplo_3.py 
correo = input("Ingresa tu correo: ")
# Extraer solo el dominio
usuario, dominio = correo.split("@")
print(f"Tu dominio es: {dominio}")
print(f"Inicial del usuario: {usuario[0]}")
Output (si ingresa [email protected])
Tu dominio es: gmail.com
Inicial del usuario: l

Ejemplo 4 — Comprobar si una palabra es palíndromo:

ejemplo_4.py 
palabra = "radar"

if palabra == palabra[::-1]:
    print("Es palíndromo")
else:
    print("No es palíndromo")
Output
Es palíndromo
04 · Práctica

5 Ejercicios para practicar

Solo usa slicing. Sin bucles ni métodos complejos por ahora.

01
Primeras y últimas letras

Pide una palabra al usuario e imprime su primera letra, su última letra, y las 3 letras del medio (desde índice 1 hasta -1).

"python" → P | n | ytho
02
¿Es palíndromo?

Pide una palabra y compara si es igual a sí misma al revés usando slicing. Muestra "Es palíndromo" o "No es palíndromo".

"radar" → Es palíndromo
03
Censurar una palabra

Pide una palabra y muéstrala con solo la primera y última letra visibles, reemplazando el medio con asteriscos. Usa slicing para obtener las letras y concatena con "*".

"secreto" → s*****o
04
Nombre abreviado

Pide al usuario su nombre completo (ej. "Juan Perez") y muestra solo la inicial del nombre seguida de un punto y el apellido completo.

"Juan Perez" → J. Perez
05
Validar extensión de archivo

Pide el nombre de un archivo y verifica si termina en ".py" usando slicing con índice negativo. Muestra "Es un archivo Python" o "No es un archivo Python".

"programa.py" → Es un archivo Python
