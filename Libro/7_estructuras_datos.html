01 · Lista

Listas [ ]

Una lista es una colección ordenada y mutable de elementos. Puedes agregar, quitar y modificar elementos después de crearla. Los elementos se separan por comas y se envuelven en corchetes.

🔵 Estructura de una lista
[
"Ana"
índice 0 / -4
"Luis"
índice 1 / -3
"Carlos"
índice 2 / -2
"María"
índice 3 / -1
]

Los índices empiezan en 0. Los negativos cuentan desde el final: -1 es el último elemento.

listas_basico.py
# Crear una lista
frutas = ["manzana", "pera", "uva"]
numeros = [10, 20, 30, 40]
mixta = ["Ana", 26, True, 3.14]   # puede tener tipos distintos
vacia = []

# Acceder por índice
print(frutas[0])    # "manzana"
print(frutas[-1])   # "uva"

# Slicing
print(numeros[1:3])  # [20, 30]

# Modificar un elemento
frutas[1] = "mango"
print(frutas)       # ["manzana", "mango", "uva"]

# Recorrer con for
for fruta in frutas:
    print(fruta)
Output
manzana
uva
[20, 30]
['manzana', 'mango', 'uva']
manzana
mango

02 · Lista

Métodos de lista

.append(x)
Agrega x al final de la lista
lista = [1, 2]
lista.append(3)
→ [1, 2, 3]
.insert(i, x)
Inserta x en la posición i
lista = [1, 3]
lista.insert(1, 2)
→ [1, 2, 3]
.remove(x)
Elimina la primera ocurrencia de x
lista = [1, 2, 2, 3]
lista.remove(2)
→ [1, 2, 3]
.pop(i)
Elimina y retorna el elemento en posición i (defecto: último)
lista = [1, 2, 3]
lista.pop()
→ retorna 3, lista=[1,2]
.sort()
Ordena la lista in-place (modifica la original)
lista = [3, 1, 2]
lista.sort()
→ [1, 2, 3]
.reverse()
Invierte el orden de la lista
lista = [1, 2, 3]
lista.reverse()
→ [3, 2, 1]
.index(x)
Retorna el índice de la primera ocurrencia de x
[10, 20, 30].index(20)
→ 1
.count(x)
Cuenta cuántas veces aparece x
[1, 2, 2, 3].count(2)
→ 2
.extend(lista2)
Agrega todos los elementos de lista2 al final
[1,2].extend([3,4])
→ [1, 2, 3, 4]
len(lista)
Retorna el número de elementos
len([1, 2, 3, 4])
→ 4
💡

sorted() vs .sort(): .sort() modifica la lista original. sorted(lista) devuelve una nueva lista ordenada sin tocar la original.

03 · Tupla

Tuplas ( )

Una tupla es una colección ordenada e inmutable. Una vez creada, no puedes cambiar sus elementos. Se usan cuando los datos no deben modificarse — coordenadas, colores RGB, configuraciones fijas.

🟣 Estructura de una tupla
(
-12.04
latitud [0]
-77.03
longitud [1]
) ← coordenadas de Lima
tuplas_basico.py
# Crear una tupla
coordenadas = (-12.04, -77.03)
rgb_rojo = (255, 0, 0)
solo_uno = (42,)   # ← la coma es obligatoria para tupla de 1 elemento

# Acceder por índice (igual que lista)
print(coordenadas[0])  # -12.04

# Desempacar (unpacking)
lat, lon = coordenadas
print(f"Lat: {lat}{lon})

# Recorrer
for valor in rgb_rojo:
    print(valor)

# ❌ Intentar modificar → TypeError
# coordenadas[0] = 0  ← esto da error
Output
-12.04
Lat: -12.04, Lon: -77.03
255
0
0
04 · Tupla

Métodos de tupla

Las tuplas tienen muy pocos métodos porque son inmutables. Solo pueden consultar, no modificar.

.count(x)
Cuenta cuántas veces aparece x
(1, 2, 2, 3).count(2)
→ 2
.index(x)
Índice de la primera ocurrencia de x
(10, 20, 30).index(20)
→ 1
len(tupla)
Número de elementos
len((1, 2, 3))
→ 3
x in tupla
True si x está en la tupla
5 in (1, 3, 5, 7)
→ True
💡

¿Por qué usar tuplas si tienen menos métodos? Precisamente porque son inmutables: son más rápidas, más seguras y le dicen al lector del código "estos datos no deben cambiar".

05 · Diccionario

Diccionarios { }

Un diccionario almacena pares clave: valor. En lugar de acceder por número, accedes por nombre. Es perfecto para representar objetos del mundo real — un usuario, un producto, una configuración.

🟢 Estructura de un diccionario
"nombre"
:
"Lian"
"edad"
:
26
"ciudad"
:
"Lima"

Las claves son únicas. Los valores pueden ser cualquier tipo.

diccionarios_basico.py
# Crear un diccionario
usuario = {
    "nombre": "Lian",
    "edad": 26,
    "ciudad": "Lima",
    "activo": True
}

# Acceder por clave
print(usuario["nombre"])   # "Lian"
print(usuario["edad"])    # 26

# Agregar o modificar
usuario["email"] = "lian@email.com"
usuario["edad"] = 27

# Eliminar
del usuario["activo"]

# Verificar si una clave existe
if "nombre" in usuario:
    print("Tiene nombre")

# Recorrer
for clave, valor in usuario.items():
    print(f"{clave}: {valor}")
Output
Lian
26
Tiene nombre
nombre: Lian
edad: 27
ciudad: Lima
email: lian@email.com
🚨

Error común: acceder a una clave que no existe lanza KeyError. Usa .get(clave) para evitarlo — retorna None si la clave no existe en lugar de lanzar un error.

06 · Diccionario

Métodos de diccionario

.get(clave, default)
Retorna el valor de la clave. Si no existe, retorna default (None por defecto)
d = {"a": 1}
d.get("b", 0)
→ 0 (no KeyError)
.keys()
Retorna todas las claves
{"a":1,"b":2}.keys()
→ dict_keys(['a','b'])
.values()
Retorna todos los valores
{"a":1,"b":2}.values()
→ dict_values([1, 2])
.items()
Retorna pares (clave, valor) — ideal para for loops
for k, v in d.items():
  print(k, v)
→ clave valor por línea
.pop(clave)
Elimina la clave y retorna su valor
d = {"a":1,"b":2}
d.pop("a")
→ retorna 1, d={"b":2}
.update(otro_dict)
Agrega o actualiza con los pares del otro dict
d = {"a":1}
d.update({"b":2})
→ {"a":1, "b":2}
len(dict)
Número de pares clave-valor
len({"a":1,"b":2,"c":3})
→ 3
clave in dict
True si la clave existe
"nombre" in usuario
→ True / False

01
Gestión de lista de compras

Crea un programa que gestione una lista de compras. El usuario puede agregar productos, eliminarlos, ver la lista completa o salir. Usa una lista para almacenar los productos.

🎯 Reto

Implementa un menú con while True con 4 opciones: agregar, eliminar, ver y salir. Al eliminar, verifica que el producto exista antes de intentar borrarlo.

💡 Pista de diseño

Usa .append() para agregar y .remove() para eliminar. Para verificar si existe, usa if producto in lista. Muestra la lista con un for.

📋 Tabla de análisis
IdentificadorDescripciónTipoFunciónRestricción
lista_comprasProductos en la listalistProcesoEmpieza vacía
opcionAcción elegida por el usuariostrEntrada"1","2","3" o "4"
productoNombre del producto a agregar/eliminarstrEntradaSolo si opción es 1 o 2
SOLUCIÓN
lista_compras.py
lista_compras = []

while True:
    print("\n1. Agregar  2. Eliminar  3. Ver lista  4. Salir")
    opcion = input("Opción: ").strip()

    if opcion == "1":
        producto = input("Producto: ").strip().lower()
        lista_compras.append(producto)
        print(f"✓ '{producto}' agregado")

    elif opcion == "2":
        producto = input("Producto a eliminar: ").strip().lower()
        if producto in lista_compras:
            lista_compras.remove(producto)
            print(f"✓ '{producto}' eliminado")
        else:
            print("Producto no encontrado")

    elif opcion == "3":
        if len(lista_compras) == 0:
            print("Lista vacía")
        else:
            for i, p in enumerate(lista_compras, 1):
                print(f"  {i}. {p}")

    elif opcion == "4":
        print("¡Hasta luego!")
        break

Contador de palabras con diccionario

Pide al usuario una frase y cuenta cuántas veces aparece cada palabra. Usa un diccionario donde las claves son las palabras y los valores son los conteos. Al final muestra las palabras ordenadas de mayor a menor frecuencia.

🎯 Reto

Recorre las palabras de la frase con un for. Para cada palabra, si ya existe en el diccionario suma 1; si no, agrégala con valor 1. Usa .get() para simplificar la lógica.

💡 Pista de diseño

conteo[palabra] = conteo.get(palabra, 0) + 1 es el patrón estándar. Luego ordena el diccionario con sorted(conteo.items(), key=lambda x: x[1], reverse=True).

📋 Tabla de análisis
IdentificadorDescripciónTipoFunciónRestricción
fraseTexto ingresado por el usuariostrEntradaNormalizar con .lower()
palabrasLista de palabras tras .split()listProcesoCada elemento es una palabra
conteoDiccionario palabra → cantidaddictProceso/SalidaEmpieza vacío {}
SOLUCIÓN
contador_palabras.py
frase = input("Ingresa una frase: ").lower().strip()
palabras = frase.split()
conteo = {}

for palabra in palabras:
    conteo[palabra] = conteo.get(palabra, 0) + 1

# Ordenar de mayor a menor frecuencia
ordenado = sorted(conteo.items(), key=lambda x: x[1], reverse=True)

print("\nPalabras más frecuentes:")
for palabra, cantidad in ordenado:
    print(f"  {palabra}: {cantidad}")

Agenda de contactos con diccionario

Crea una agenda donde cada contacto tiene nombre, teléfono y email. El usuario puede agregar contactos, buscar por nombre y listar todos. Usa un diccionario de diccionarios.

🎯 Reto

La estructura principal es agenda = {} donde cada clave es el nombre y el valor es otro diccionario con {"tel": ..., "email": ...}. Usa .get() para buscar sin errores.

💡 Pista de diseño

Para agregar: agenda[nombre] = {"tel": tel, "email": email}. Para buscar: contacto = agenda.get(nombre) — si es None, no existe. Para listar: recorre agenda.items().

📋 Tabla de análisis
IdentificadorDescripciónTipoFunciónRestricción
agendaDiccionario principal nombre→datosdictProcesoEmpieza vacío
nombreClave del contactostrEntradaNormalizar con .title()
contactoSub-diccionario con tel y emaildictProceso/Salida{"tel":..., "email":...}
SOLUCIÓN
agenda.py
agenda = {}

while True:
    print("\n1. Agregar  2. Buscar  3. Listar  4. Salir")
    op = input("Opción: ")

    if op == "1":
        nombre = input("Nombre: ").strip().title()
        tel    = input("Teléfono: ").strip()
        email  = input("Email: ").strip().lower()
        agenda[nombre] = {"tel": tel, "email": email}
        print(f"✓ {nombre} agregado")

    elif op == "2":
        nombre = input("Buscar: ").strip().title()
        c = agenda.get(nombre)
        if c:
            print(f"Tel: {c['tel']} | Email: {c['email']}")
        else:
            print("No encontrado")

    elif op == "3":
        if not agenda:
            print("Agenda vacía")
        else:
            for nombre, datos in agenda.items():
                print(f"{nombre}: {datos['tel']} | {datos['email']}")

    elif op == "4":
        break

Crea una función que reciba el nombre de un alumno y una tupla de notas. Calcula y muestra el promedio, nota máxima, nota mínima y si aprobó. Pruébala con al menos 3 alumnos almacenados en una lista.

Usa una lista de tuplas donde cada tupla es (nombre, notas_tupla). La función analizar(nombre, notas) calcula las estadísticas. Recorre la lista con un for y llama a la función por cada alumno.

📋 Tabla de análisis
IdentificadorDescripciónTipoFunciónRestricción

estadisticas_notas.py
def analizar(nombre, notas):
    prom = sum(notas) / len(notas)
    estado = "✓ Aprobado" if prom >= 60 else "✗ Reprobado"
    print(f"\n{nombre}")
    print(f"  Notas:    {notas}")
    print(f"  Promedio: {prom:.1f}")
    print(f"  Máxima:   {max(notas)}")
    print(f"  Mínima:   {min(notas)}")
    print(f"  Estado:   {estado}")

alumnos = [
    ("Ana",    (85, 90, 78, 92)),
    ("Luis",   (55, 60, 58, 62)),
    ("Carlos", (95, 88, 97, 91)),
]

for nombre, notas in alumnos:
    analizar(nombre, notas)

Inventario de tienda

Crea un inventario de tienda usando un diccionario donde cada producto tiene precio y stock. El usuario puede ver el inventario, comprar productos (reduce el stock) y ver el valor total del inventario.

🎯 Reto

El inventario es un diccionario: {"manzana": {"precio": 2.5, "stock": 100}}. Al comprar, verifica que haya suficiente stock. El valor total es la suma de precio × stock de todos los productos.

💡 Pista de diseño

Para calcular el total, recorre inventario.values() y acumula producto["precio"] * producto["stock"]. Para comprar, accede con inventario[nombre]["stock"] y resta la cantidad.

📋 Tabla de análisis
IdentificadorDescripciónTipoFunciónRestricción
inventarioDict principal producto→datosdictProcesoPredefinido con datos iniciales
productoSub-dict {"precio":…,"stock":…}dictProcesostock no puede ser negativo
totalValor total del inventariofloatSalidasuma de precio×stock
SOLUCIÓN
inventario.py
inventario = {
    "manzana": {"precio": 2.50, "stock": 100},
    "pera":    {"precio": 3.00, "stock": 50},
    "uva":     {"precio": 5.50, "stock": 30},
}

while True:
    print("\n1. Ver inventario  2. Comprar  3. Valor total  4. Salir")
    op = input("Opción: ")

    if op == "1":
        print(f"\n{'Producto':<12} {'Precio':>8} {'Stock':>8}")
        print("-" * 30)
        for nombre, datos in inventario.items():
            print(f"{nombre:<12} S/{datos['precio']:>6.2f} {datos['stock']:>8}")

    elif op == "2":
        nombre = input("Producto: ").strip().lower()
        if nombre not in inventario:
            print("Producto no existe")
        else:
            cantidad = int(input("Cantidad: "))
            if cantidad > inventario[nombre]["stock"]:
                print(f"Stock insuficiente ({inventario[nombre]['stock']} disponibles)")
            else:
                inventario[nombre]["stock"] -= cantidad
                total = inventario[nombre]["precio"] * cantidad
                print(f"✓ Compra: S/{total:.2f}")

    elif op == "3":
        total = sum(d["precio"] * d["stock"] for d in inventario.values())
        print(f"Valor total del inventario: S/{total:.2f}")

    elif op == "4":
        break
