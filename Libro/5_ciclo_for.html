El Ciclo for en Python

Una guía sencilla con ejemplos de la vida cotidiana y código real

01 · Concepto

¿Qué es un ciclo for?

Un ciclo for le dice a Python: "repite estas instrucciones, una vez por cada elemento de esta lista". Cuando se acaban los elementos, el ciclo termina solo.

Piénsalo así: el for es como un asistente que toma una lista y procesa cada ítem uno por uno, sin que tú tengas que decirle cuántas veces repetir.

02 · Analogías

En la vida cotidiana

El ciclo for aparece constantemente en situaciones del día a día:

🛒

Lista del supermercado

Por cada producto en la lista → buscarlo en la tienda → ponerlo en el carrito.

📬

Repartidor de correo

Por cada carta en la bolsa → leer la dirección → dejarla en la puerta correcta.

🎵

Playlist de música

Por cada canción en la playlist → reproducirla → pasar a la siguiente.

📝

Corregir exámenes

Por cada examen en la pila → revisar las respuestas → asignar una nota.

03 · Estructura

Anatomía del ciclo for

Todo ciclo for tiene la misma forma:

for elemento in secuencia :
↓ se ejecuta una vez por cada elemento
···· instrucción
palabra reservada
variable temporal
lista / string / rango
lo que se repite

La variable temporal (puedes llamarla como quieras) toma el valor de cada elemento en cada vuelta. El indentado (los espacios al inicio) le dice a Python qué líneas pertenecen al ciclo.

04 · Ejemplos

Ejemplos de código

Ejemplo 1 — Recorrer una lista de nombres:

ejemplo_1.py 
amigos = ["Ana", "Luis", "Carlos"]

for amigo in amigos:
    print(f"Hola, {amigo}!")
Output
Hola, Ana!
Hola, Luis!
Hola, Carlos!

Ejemplo 2 — Recorrer las letras de una palabra:

ejemplo_2.py 
palabra = "python"

for letra in palabra:
    print(letra)
Output
p
y
t
h
o
n

Ejemplo 3 — Usar range() para repetir N veces:

ejemplo_3.py 
# range(5) genera los números 0, 1, 2, 3, 4
for i in range(5):
    print(f"Vuelta número {i + 1}")
Output
Vuelta número 1
Vuelta número 2
Vuelta número 3
Vuelta número 4
Vuelta número 5

Ejemplo 4 — Acumular un resultado (contador):

ejemplo_4.py 
# Contar cuántas letras 'a' tiene una palabra
palabra = "anaconda"
contador = 0

for letra in palabra:
    if letra == "a":
        contador = contador + 1

print(f"Hay {contador} letras 'a'")
Output
Hay 3 letras 'a'
05 · Reglas

Reglas clave para recordar

⚙️

El : es obligatorio

Después de in secuencia siempre va dos puntos. Sin ellos Python da error.

↔️

El indentado importa

Las líneas dentro del for deben tener 4 espacios al inicio. Sin ellos Python no sabe qué pertenece al ciclo.

🔢

range() empieza en 0

range(5) genera 0, 1, 2, 3, 4 — cinco números, pero empieza desde cero, no desde uno.

📦

Acumula antes del for

Si necesitas un contador o suma, inicialízalo en 0 antes del ciclo, no adentro.
