01

for vs while — ¿cuándo usar cada uno?

Ambos repiten código, pero tienen un propósito distinto:

Bucle for

Sabes cuántas veces

Recorre una secuencia conocida: una lista, un string, un rango de números.

for letra in "python":
  print(letra)

# Sabe que son 6 vueltas
Bucle while

No sabes cuántas veces

Repite hasta que algo cambie. No sabe de antemano cuántas vueltas dará.

while intento != secreto:
  intento = input("?")

# Depende del usuario
💡

Regla práctica: si puedes contar las repeticiones antes de empezar → usa for. Si dependes de una condición que puede cambiar en cualquier momento → usa while.

02

Anatomía del ciclo while

while condición :
instrucciones
palabra reservada
expresión True/False
lo que se repite (indentado)

El ciclo evalúa la condición antes de cada vuelta. Si es True, ejecuta las instrucciones. Si es False, se detiene y el programa continúa.

⚠️

Importante: algo dentro del bucle debe eventualmente hacer que la condición se vuelva False. Si no, el bucle corre para siempre.

n = 1

    n = n + 1   # ← sin esto, loop infinito

Ejemplo 2 — Validar input del usuario

El uso más común del while: seguir pidiendo datos hasta que el usuario ingrese algo válido.

validar_input.py
edad = int(input("Ingresa tu edad: "))

while edad < 0 or edad > 120:
    print("Edad inválida. Intenta de nuevo.")
    edad = int(input("Ingresa tu edad: "))

print(f"Edad registrada: {edad}")
Output (si ingresa -5, luego 200, luego 25)

Edad inválida. Intenta de nuevo.

Ejemplo 3 — Adivina el número

adivina.py
secreto = 42
adivinado = False

while adivinado == False:
    intento = int(input("Adivina: "))
    if intento == secreto:
        print("¡Correcto!")
        adivinado = True
    elif intento > secreto:

05

break y continue

Dos palabras reservadas que controlan el flujo dentro de cualquier bucle:

break
Sale del bucle inmediatamente, sin importar si la condición sigue siendo True.
while True:
  n = int(input(":"))
  if n == 0:
    break # sale aquí
print("Listo")
continue
Salta el resto de la vuelta actual y vuelve a evaluar la condición.
while n < 10:
  n += 1
  if n % 2 == 0:
    continue # salta pares
  print(n) # solo impares

break — ejemplo completo

break_ejemplo.py
# Pedir números hasta que el usuario escriba "salir"
suma = 0

while True:
    entrada = input("Número (o 'salir'): ")
    if entrada == "salir":
        break
    suma = suma + int(entrada)

print(f"Suma total: {suma}")
Output (ingresó 3, 7, 2, salir)
Suma total: 12

continue — ejemplo completo

continue_ejemplo.py
# Imprimir solo números impares del 1 al 10
n = 0

while n < 10:
    n += 1
    if n % 2 == 0:
        continue   # salta los pares, vuelve al while
    print(n)
Output
1
3
5
7
9
06

while True — el bucle infinito controlado

while True es un patrón muy común en Python. La condición siempre es True, así que el bucle nunca termina por sí solo — depende de un break interno para salir.

Es útil cuando quieres que el bucle corra al menos una vez antes de evaluar si debe continuar:

while_true.py
# Pedir contraseña hasta que sea correcta
while True:
    password = input("Contraseña: ")
    if password == "python123":
        print("Acceso concedido ✓")
        break
    print("Contraseña incorrecta, intenta de nuevo.")
Output
Contraseña incorrecta, intenta de nuevo.  (ingresó "1234")
Contraseña incorrecta, intenta de nuevo.  (ingresó "qwerty")
Acceso concedido ✓                        (ingresó "python123")
💡

while True vs while condición: usa while True + break cuando la condición de salida se evalúa después de ejecutar las instrucciones. Usa while condición cuando la evaluación va antes.

07

while / else — el caso especial de Python

Python permite agregar un bloque else al while. El else se ejecuta solo si el bucle terminó normalmente (la condición se volvió False), no si salió con un break.

while / else
El else corre si el while terminó sin break. Si hubo break, el else se salta.
while condicion:
  ...
  if algo: break
else:
  print("terminó sin break")
while_else.py
# Buscar un número negativo en una lista de inputs
intentos = 3

while intentos > 0:
    n = int(input("Ingresa un número: "))
    if n < 0:
        print("Encontraste un negativo. ¡Lograste!")
        break
    intentos -= 1
else:
    print("Se acabaron los intentos sin encontrar negativos.")
Output A (encontró negativo)
Encontraste un negativo. ¡Lograste!
Output B (se acabaron los intentos)
Se acabaron los intentos sin encontrar negativos.
08

⚠️ El bucle infinito — el error más común

🚨

Un bucle infinito ocurre cuando la condición del while nunca se vuelve False. El programa se congela y nunca termina. En Jupyter puedes pararlo con el botón ⏹ o presionando I, I (dos veces la tecla I).

Causas comunes

infinito_error.py
# ❌ ERROR 1: olvidaste actualizar la variable
n = 1
while n <= 5:
    print(n)
    # n nunca cambia → loop infinito

# ❌ ERROR 2: la condición siempre es True
x = 10
while x > 0:
    print(x)
    x += 1   # x crece → nunca llega a 0

# ❌ ERROR 3: break dentro de un if que nunca se cumple
while True:
    n = int(input(":"))
    if n == "salir":  # comparando int con str → nunca True
        break
infinito_correcto.py
# ✅ CORRECTO: la variable se actualiza en cada vuelta
n = 1
while n <= 5:
    print(n)
    n += 1   # ← esto hace que eventualmente n > 5
Output
1
2
3
4
5

Ejemplos reales combinados

Ejemplo — Menú de opciones

Un patrón clásico: mostrar un menú y repetir hasta que el usuario elija salir.

menu.py
while True:
    print("\n1. Saludar")
    print("2. Despedirse")
    print("3. Salir")
    opcion = input("Elige: ")

    if opcion == "1":
        print("¡Hola!")
    elif opcion == "2":
        print("¡Adiós!")
    elif opcion == "3":
        print("Saliendo...")
        break
    else:
        print("Opción inválida")

Ejemplo — Contador de intentos

intentos.py
secreto = 42
intentos = 0
max_intentos = 5

while intentos < max_intentos:
    n = int(input(f"Intento {intentos + 1}/{max_intentos}: "))
    intentos += 1
    if n == secreto:
        print(f"¡Correcto en {intentos} intento/s!")
        break
    elif n > secreto:
        print("Muy alto")
    else:
        print("Muy bajo")
else:
    print(f"Sin intentos. El número era {secreto}")
