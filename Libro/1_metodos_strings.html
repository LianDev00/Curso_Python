01

¿Qué es un método?

Un método es una función que pertenece a un tipo de dato. En Python, los strings tienen sus propios métodos que puedes llamar con un punto después de la variable.

concepto.py 
texto = "hola mundo"

# variable . método ( )
texto.upper()     # → "HOLA MUNDO"
texto.capitalize() # → "Hola mundo"
💡

Los métodos no modifican la variable original — devuelven una nueva cadena. Si quieres guardar el resultado, asígnalo: texto = texto.upper()

02

Transformar texto

Estos métodos cambian las mayúsculas y minúsculas del texto.

.upper()
Convierte todo a MAYÚSCULAS
"hola".upper()
→ "HOLA"
.lower()
Convierte todo a minúsculas
"HOLA".lower()
→ "hola"
.capitalize()
Primera letra mayúscula, el resto minúscula
"hola MUNDO".capitalize()
→ "Hola mundo"
.title()
Primera letra de cada palabra en mayúscula
"hola mundo".title()
→ "Hola Mundo"
.swapcase()
Invierte mayúsculas y minúsculas
"Hola Mundo".swapcase()
→ "hOLA mUNDO"
transformar.py 
nombre = input("Tu nombre: ")

print(nombre.upper())      # para gritar
print(nombre.lower())      # normalizar antes de comparar
print(nombre.title())      # formato de nombre propio
Output (si ingresa "lian")
LIAN
lian
Lian
03

Limpiar texto

Eliminan espacios o caracteres no deseados, muy útil al procesar input del usuario.

.strip()
Elimina espacios al inicio y al final
" hola ".strip()
→ "hola"
.lstrip()
Elimina espacios solo al inicio (izquierda)
" hola ".lstrip()
→ "hola "
.rstrip()
Elimina espacios solo al final (derecha)
" hola ".rstrip()
→ " hola"
💡

Es buena práctica usar .strip().lower() juntos al recibir input, así evitas errores por espacios accidentales o mayúsculas.

04

Buscar y reemplazar

.find(x)
Devuelve el índice de la primera aparición de x. Si no lo encuentra, devuelve -1.
"python".find("t")
→ 2
.count(x)
Cuenta cuántas veces aparece x en la cadena
"banana".count("a")
→ 3
.replace(a, b)
Reemplaza todas las ocurrencias de a por b
"hola mundo".replace("o","0")
→ "h0la mund0"
.startswith(x)
True si la cadena empieza con x
"python".startswith("py")
→ True
.endswith(x)
True si la cadena termina con x
"foto.jpg".endswith(".jpg")
→ True
x in cadena
True si x está contenido en la cadena (no es método, es operador)
"py" in "python"
→ True
05

Verificar contenido

Estos métodos devuelven True o False según el contenido de la cadena. Son muy útiles dentro de condicionales y bucles.

.isalpha()
True si todos los caracteres son letras
"hola".isalpha()
→ True
.isdigit()
True si todos los caracteres son dígitos
"1234".isdigit()
→ True
.isalnum()
True si todos son letras o números (sin espacios ni símbolos)
"abc123".isalnum()
→ True
.isspace()
True si todos los caracteres son espacios
" ".isspace()
→ True
.isupper()
True si todas las letras son mayúsculas
"HOLA".isupper()
→ True
.islower()
True si todas las letras son minúsculas
"hola".islower()
→ True
verificar.py 
oracion = "Hola Mundo 123"
mayus = 0
minus = 0
digitos = 0

for c in oracion:
    if c.isupper():
        mayus += 1
    elif c.islower():
        minus += 1
    elif c.isdigit():
        digitos += 1

print(f"Mayúsculas: {mayus}, Minúsculas: {minus}, Dígitos: {digitos}")
Output
Mayúsculas: 2, Minúsculas: 8, Dígitos: 3
06

Dividir y unir

.split(separador)

Divide una cadena en una lista de partes, cortando donde encuentre el separador. Si no le pasas separador, corta por espacios.

split.py 
oracion = "hola como estas"
partes = oracion.split()
print(partes)       # ['hola', 'como', 'estas']
print(partes[0])    # 'hola'
print(partes[-1])   # 'estas'

# Con separador específico
fecha = "24/03/2026"
dia, mes, anio = fecha.split("/")
print(dia, mes, anio)  # 24 03 2026
Output
['hola', 'como', 'estas']
hola
estas
24 03 2026

.join(lista)

Es lo opuesto a split() — une una lista de strings en una sola cadena, poniendo el separador entre cada elemento.

join.py 
palabras = ["hola", "como", "estas"]

print(" ".join(palabras))    # une con espacio
print("-".join(palabras))    # une con guión
print(" → ".join(palabras))  # une con flecha

# También funciona con strings directamente
letras = "python"
print(" + ".join(letras))  # une cada letra
Output
hola como estas
hola-como-estas
hola → como → estas
p + y + t + h + o + n
⚠️

En join() el separador va antes del paréntesis, no adentro. Es " ".join(lista), no lista.join(" "). Al revés no funciona.

07

ord() y chr()

Estas no son métodos de string sino funciones integradas de Python. Convierten entre caracteres y sus números ASCII.

ord(carácter)
Devuelve el número ASCII del carácter
ord("a")
→ 97
chr(número)
Devuelve el carácter que corresponde al número ASCII
chr(97)
→ "a"

Los números ASCII más útiles para recordar:

ascii.py 
# Minúsculas: a=97 ... z=122
# Mayúsculas: A=65 ... Z=90
# Dígitos:    0=48 ... 9=57

# Cifrado César: desplazar letra 3 posiciones
letra = "a"
nueva = chr(ord(letra) + 3)
print(nueva)   # → "d"

# Ver el código de cualquier carácter
for c in "abc":
    print(f"{c} = {ord(c)}")
Output
d
a = 97
b = 98
c = 99

Ejemplos combinados

Ejemplo 1 — Limpiar y normalizar input del usuario:

normalizar.py 
correo = input("Tu correo: ").strip().lower()

if "@" in correo:
    usuario, dominio = correo.split("@")
    print(f"Usuario: {usuario}")
    print(f"Dominio: {dominio}")
else:
    print("Correo inválido")

Ejemplo 2 — Contar vocales con for + isalpha:

vocales.py 
texto = input("Texto: ").lower()
vocales = 0

for c in texto:
    if c in "aeiou":
        vocales += 1

print(f"{vocales} vocales")

Ejemplo 3 — Cifrado César completo:

cesar.py 
palabra = input("Palabra: ").lower()
n = int(input("Desplazamiento: "))
resultado = ""

for c in palabra:
    if c.isalpha():
        resultado += chr(ord(c) + n)
    else:
        resultado += c   # espacios y símbolos sin cambio

print(resultado)
Output (palabra="hola", n=3)
