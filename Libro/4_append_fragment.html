Método de Lista

.append() en Python

.append() es un método de las listas, no de los strings. Agrega un nuevo elemento al final de una lista existente.

A diferencia de los métodos de string como .lower() o .strip(), append() sí modifica la lista original — no devuelve una nueva.

Es como construir una fila: cada vez que llamas a append(), pones un elemento nuevo al final de la fila.

01 · Uso básico

Agregar elementos a una lista

append_basico.py
frutas = []                   # lista vacía

frutas.append("manzana")
frutas.append("pera")
frutas.append("uva")

print(frutas)
print(frutas[0])    # primer elemento
print(frutas[-1])   # último elemento
Output
['manzana', 'pera', 'uva']
manzana
uva
02 · Con bucle for

Acumular resultados dentro de un bucle

Este es el uso más común — ir guardando resultados mientras recorres algo:

append_for.py
# Guardar solo las vocales de una palabra
palabra = "murciélago"
vocales_encontradas = []

for c in palabra:
    if c in "aeiouáéíóú":
        vocales_encontradas.append(c)

print(vocales_encontradas)
print(len(vocales_encontradas), "vocales")
Output
['u', 'i', 'é', 'a', 'o']
5 vocales
03 · Caso real — suma acumulada

Guardar resultados parciales para unirlos con join()

Aquí está exactamente el patrón del ejercicio 3:

suma_acumulada.py
entrada = input("Números: ")    # ej: "3 5 2 8"
numeros = entrada.split()
acumulada = 0
resultados = []               # guardamos cada suma parcial aquí

for n in numeros:
    acumulada = acumulada + int(n)
    resultados.append(str(acumulada))  # str() porque join() necesita strings

print(" → ".join(resultados))
Output (si ingresa "3 5 2 8")
3 → 8 → 10 → 18
💡

¿Por qué str(acumulada)? Porque join() solo funciona con listas de strings, no de números. Por eso conviertes el número a string antes de guardarlo con append().

04 · append() vs +=

¿Cuándo usar cada uno?

append() → para listas
items = []
items.append("a")
items.append("b")
# ['a', 'b']

Úsalo cuando al final necesitas join() con un separador como " → " o "-".

+= → para strings
texto = ""
texto += "a"
texto += "b"
# "ab"

Úsalo cuando construyes texto directo sin separador, como en el cifrado César.
